# Movie Ticket Booking System

## Overview

A full-stack Movie Ticket Booking System that allows users to browse movies, search movies, book tickets, authenticate using JWT, and receive booking confirmation emails.

## Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS

### Backend

* Spring Boot
* Spring Security
* JWT Authentication
* Spring Data JPA

### Database

* MySQL

### Email Service

* JavaMailSender

## Features

### User Features

* Browse movies
* Search movies
* Book movie tickets
* User authentication
* Receive booking confirmation emails

### Admin Features

* Manage movies
* View bookings
* Manage users
